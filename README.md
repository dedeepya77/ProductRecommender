# 🛍️ Smart Product Recommender (Startup MVP)

An AI-powered product recommendation web application built using **Streamlit**, **Machine Learning (TF-IDF + Cosine Similarity)**, and Python.

This project simulates a **mini e-commerce platform** with features like login/signup, product search, recommendations, cart system, and order tracking.

---

## 🌐 Live Demo
👉 https://appuctrecommender-8ttav7aojsu7bx6plm9igv.streamlit.app/

---

## 📌 Features

### 🧠 AI Recommendation System
- Content-based filtering using TF-IDF
- Cosine similarity for product matching
- Smart recommendations based on product name, category, and features

### 🛒 E-Commerce Features
- Product browsing system
- Add to cart / remove from cart
- Checkout simulation
- Order history tracking

### 🔐 Authentication
- Simple login & signup system
- Session-based user handling

### 🔎 Search & UX
- Live product suggestions
- Filter by category
- Price filtering
- Clean UI with Streamlit
